As a Cooking student, I want to see a list of all available cooking schools nearby.
As a Cooking student, I want to select a cooking school and see its details such as location, address, name, and rating.
As a Cooking student, I want to submit a meal plan approval request for a selected cooking school, so that a chef instructor can review my planned recipes before I start cooking.
As a Cooking student, I want to add recipes to my meal plan approval request, so that the instructor can see the cuisine type, difficulty, required ingredients, and estimated preparation time.
As a Chef Instructor, I want to open a workflow task UI for pending meal plan approval requests, so that I can approve the request, reject it, or ask the student to revise the plan.
As a Chef Instructor, I want approved meal plan requests to update their status to Approved, so that students know they can start preparing the recipes.
As a Chef Instructor, I want rejected meal plan requests to update their status to Rejected with review notes, so that students understand what needs to change.
As a Cooking admin, I want to create and manage recipes, so that students can select recipes from an approved recipe catalog.
As a Cooking admin, I want to create and manage cooking schools, so that students can request meal plan approval for the correct location.
