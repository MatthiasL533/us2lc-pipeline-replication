# Reference Model Inputs

This directory contains the reference materials used for the domain-model comparison in the thesis.

- `gold_silver_standard.xlsx`: Visual Narrator 2.0 gold/silver standard workbook used for the reused `camperplus`, `fish_chips`, `grocery`, and `sports` datasets.
- `brewery_domain_model.json`: extracted Mendix reference domain model for `brewery`.
- `education_domain_model.json`: extracted Mendix reference domain model for `education`.
- `cinema_domain_model.json`: extracted Mendix reference domain model for `cinema`.
- `collaboration_domain_model.json`: extracted Mendix reference domain model for `collaboration`.
- `matching_domain_model.json`: extracted Mendix reference domain model for `matching`.

The quantitative scoring workbooks in `../quantitative_analysis/` contain the manual matching and computed precision/recall tables derived from these references.
